📁 Dataset Note
This project is based on the publicly available PlantVillage dataset, originally sourced from:

Kaggle Dataset: PlantVillage Dataset by AbdallahAlidev

Download Link-
https://www.kaggle.com/datasets/abdallahalidev/plantvillage-dataset

🔸 Original Dataset Info:

Contains 38 plant disease and healthy leaf classes.


🔹 Modifications for This Project:

Only 24 classes were selected from the original 38.

Each class was balanced to 500 images to maintain uniformity across categories and reduce overall size.

The selected class labels used in this project are:
